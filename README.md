# 🏥 Hospital Management System (SQL Project)

## 📌 Project Overview
This project is a **Hospital Management System** built using SQL. It manages hospital data such as patients, doctors, appointments, procedures, and billing.

It demonstrates:
- Database design (ER Model)
- SQL querying (Basic → Advanced)
- Data analysis using SQL

---

## 🗂️ Database Schema

The system includes the following tables:

- **Patients** – Stores patient details  
- **Doctors** – Stores doctor information  
- **Appointments** – Connects patients and doctors  
- **Procedures** – Stores treatment details  
- **Billing** – Stores transaction and revenue data

## 🔍 Key SQL Concepts Covered

### 🔹 Basic Queries
- SELECT, WHERE, ORDER BY  

### 🔹 Intermediate Queries
- JOINs  
- GROUP BY  
- Aggregations (SUM, AVG, COUNT)  

### 🔹 Advanced Queries
- Subqueries  
- Window Functions (RANK, cumulative sum)  

### 🔹 Interview Questions
- INNER JOIN vs LEFT JOIN  
- Foreign Key Constraints  
- Normalization  
- NULL Handling
